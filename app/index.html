H1 - PRIMARY PAGE TITLE

The first SCSS file you should probably look at is _vars.scss. Set primary site style variables (fonts, font sizes, colors, anything used repeatedly) in the _vars.scss file, which will then cascade down to both the TEMPLATE and MAIN SCSS files.

Custom, project-specific changes should be done in the MAIN SCSS files, and the TEMPLATE SCSS files should largely be left alone and overridden in the MAIN files. The TEMPLATE SCSS files are intended to be the underlying framework of this starter project, with the MAIN files handling any code that is specific to the job you're working. If you have a problem or suggestion, reach out to areimel@thoriumdigital.com.

This starter project is meant to be unopinionated (ie, not designed with any particular project in mind), while also providing you with as much support and features as possible. Also, there's a cute red panda.

NOTES and BEST PRACTICES:

  • Background Images: set the background-image src as inline CSS, and set the rest in the CSS files.
  • Use class names, not elements, in your CSS selectors. This lets you easily change the element while leaving styling unaffected.
  • Try and keep your CSS selectors short and readable - use a parent class for the whole element while giving it's components verbose and descriptive names.
  • Keep good comments and documentation. Write your code as if the next dev will have no prior knowledge of the project, and to let them jump in with minimal guidence.
